SERVILABS · Privacidad

Política de Privacidad

Última actualización: 12 de julio de 2026.

SERVILABS es una plataforma para conectar clientes con profesionales y prestadores de servicios. Esta política explica qué datos usamos, para qué los usamos y cómo protegemos a las personas dentro de la app.

1. Datos que podemos recopilar

  • Datos de cuenta: nombre, email, teléfono, rol, ubicación declarada y datos necesarios para operar el perfil.
  • Datos profesionales: rubros, experiencia, tarifa base, radio de trabajo, certificados y preferencias laborales.
  • Solicitudes y mensajes: publicaciones de trabajo, presupuestos, estados, coordinación y chat interno.
  • Ubicación: coordenadas o ubicación aproximada para mostrar profesionales cercanos, calcular distancias y mejorar la coordinación.
  • Notificaciones: identificadores técnicos de suscripción push para avisar sobre trabajos, mensajes o cambios de estado.
  • Pagos: datos operativos de órdenes, estados de pago y comprobantes necesarios para registrar transacciones.

2. Para qué usamos los datos

  • Crear y administrar cuentas de clientes, profesionales y administradores.
  • Publicar solicitudes de trabajo y conectarlas con profesionales compatibles.
  • Enviar alertas de oportunidades laborales, mensajes y cambios de estado.
  • Facilitar pagos protegidos, auditoría operativa y prevención de fraude.
  • Mejorar seguridad, trazabilidad y experiencia de uso.

3. Ubicación

La app puede solicitar acceso a ubicación para calcular distancia entre clientes y profesionales, mostrar mapas, ordenar oportunidades cercanas y facilitar la navegación hacia una solicitud o servicio. La persona usuaria puede aceptar o rechazar este permiso desde el sistema operativo.

4. Notificaciones

SERVILABS puede solicitar permiso para enviar notificaciones push. Se usan para avisar a profesionales sobre nuevas oportunidades compatibles con su rubro, y a clientes sobre mensajes o cambios de estado. La persona usuaria puede desactivar las notificaciones desde la configuración del dispositivo o la app.

5. Pagos y seguridad

Los pagos pueden procesarse mediante proveedores externos como Mercado Pago u otros procesadores integrados. SERVILABS almacena estados operativos de pago, pero no debe almacenar datos completos de tarjetas. La plataforma puede aplicar controles antifraude y auditoría para proteger operaciones con dinero.

6. Inteligencia artificial

La app puede utilizar IA para orientar rubros laborales, responder consultas y asistir en la experiencia. Las respuestas son de apoyo y no reemplazan decisiones profesionales, legales, médicas o financieras. En funciones como "La cosa del cosito", la persona usuaria puede adjuntar una foto de referencia para que ASI ayude a redactar mejor una solicitud; esa orientación no constituye diagnóstico técnico definitivo.

7. Conservación y protección

Conservamos la información necesaria para operar la plataforma, cumplir obligaciones, investigar incidentes y sostener la seguridad. Aplicamos controles de acceso, cifrado en tránsito y separación de secretos mediante variables de entorno.

8. Derechos de las personas usuarias

Las personas pueden solicitar acceso, actualización o baja de sus datos cuando corresponda. Para ejercer estos derechos o realizar consultas, escribir a: soporte@servilabs.app.

9. Cambios

Esta política puede actualizarse para reflejar cambios legales, técnicos o de producto. La versión vigente estará disponible públicamente en esta página.
